# sparkle-shop-cloud

`sparkle-shop-cloud` 是按开发文档整理出来的全新微服务空骨架，当前只完成目录结构、父 POM、公共模块、网关模块、核心业务服务模块和基础部署目录，不包含具体业务实现。

## 当前结构

- `sparkle-common`
  - `common-core`
  - `common-security`
  - `common-redis`
  - `common-web`
  - `common-log`
- `sparkle-gateway`
  - `gateway-service`
- `sparkle-services`
  - `user-service`
  - `product-service`
  - `cart-service`
  - `order-service`
  - `stock-service`
  - `coupon-service`
- `docker`
